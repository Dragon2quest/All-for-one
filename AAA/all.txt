
## PROGRAM 1: Setting Up and Basic Commands

### Aim

Initialize a Git repository, create a file, add it, and commit.

### Commands

```bash
mkdir my_project
cd my_project
git init
```

Create a file:

```bash
touch file1.txt
```

Add file to staging:

```bash
git add file1.txt
```

Commit:

```bash
git commit -m "Initial commit"
```

## PROGRAM 2: Creating and Managing Branches

### Aim

Create a branch, switch branches, and merge.

### Commands

Create and switch to new branch:

```bash
git checkout -b feature-branch
```

Switch back to master:

```bash
git checkout master
```

Merge feature branch:

```bash
git merge feature-branch
```

Delete branch (optional):

```bash
git branch -d feature-branch
```

---

## PROGRAM 3: Stashing Changes

### Aim

Save changes temporarily, switch branch, and reapply.

### Commands

Stash changes:

```bash
git stash
```

Switch branch:

```bash
git checkout another-branch
```

Apply stash:

```bash
git stash apply
```

Apply and remove stash in one step:

```bash
git stash pop
```

## PROGRAM 4: Clone a Remote Repository

### Aim

Clone a GitHub repository.

### Command

```bash
git clone https://github.com/user/repo.git
```

Using SSH:

```bash
git clone git@github.com:user/repo.git
```

## PROGRAM 5: Fetch and Rebase

### Aim

Update local branch using rebase.

### Commands

Fetch changes:

```bash
git fetch origin
```

Rebase:

```bash
git rebase origin/main
```

Continue rebase:

```bash
git rebase --continue
```

Abort rebase:

```bash
git rebase --abort
```

Force push (âš  use carefully):

```bash
git push origin main --force
```

## PROGRAM 6: Merge with Custom Commit Message

### Command

```bash
git merge feature-branch -m "Merged feature branch"
```

Using no fast-forward:

```bash
git merge feature-branch --no-ff
```

## PROGRAM 7: Git Tags

### Aim

Create a lightweight tag.

### Commands

Tag latest commit:

```bash
git tag v1.0
```

Tag specific commit:

```bash
git tag v1.0 commit_hash
```

Push tag:

```bash
git push origin v1.0
```

## PROGRAM 8: Cherry-Pick Commits

### Command

```bash
git cherry-pick start_commit^..end_commit
```

Continue cherry-pick:

```bash
git cherry-pick --continue
```

Abort:

```bash
git cherry-pick --abort
```

## PROGRAM 9: View Commit Details

### Command

```bash

```

Short info:

```bash

```

## PROGRAM 10: List Commits by Author and Date

### Command

```bash

```

## PROGRAM 11: Display Last Five Commits
